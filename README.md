## 🔍 Overview

<a href="#betterTouchTool">BetterTouchTool</a> serves Mac power users who have outgrown Apple's default input configuration, developers who want keyboard and gesture shortcuts for development workflows, creative professionals who want application-specific Touch Bar and trackpad controls, and anyone who wants to customize their Mac's input behavior beyond what System Preferences provides.

The <a href="#betterTouchTool">developer workflow use case</a> assigns custom gestures to IDE operations — three-finger tap to run tests, corner swipe to toggle the terminal, custom keyboard shortcuts for code navigation — creating a gesture-driven development environment where common operations are always one gesture away. The <a href="#betterTouchTool">creative professional workflow</a> uses application-specific Touch Bar and gesture configurations for video editing, audio production, graphic design, and other creative applications — context-appropriate input controls that match each application's specific operation vocabulary.

The <a href="#betterTouchTool">window management power user</a> uses BetterTouchTool's snap areas and gesture-triggered window resizing as a complete tiling window manager — creating specific window arrangements through gestures and snap zones rather than keyboard shortcut combinations or dedicated tiling applications. <a href="#betterTouchTool">Setapp availability</a> includes BetterTouchTool in the Setapp subscription — subscribers access BetterTouchTool without additional purchase as part of the Mac application collection.

<a href="#betterTouchTool">Haptic feedback customization</a> configures the trackpad's haptic response for BTT's custom gestures — matching the haptic feel of custom gestures to the feedback that standard macOS gestures provide for a consistent physical experience. <a href="#betterTouchTool">Remote control support</a> maps Siri Remote, Apple TV Remote, and other remote controls to custom Mac actions — controlling the Mac from a distance with the remote as a macro controller.

<a href="#betterTouchTool">AppleScript and shell script integration</a> runs custom scripts as the action for any trigger — the complete power of AppleScript and shell scripting available as the result of any gesture, shortcut, or button press. <a href="#betterTouchTool">Active development by Andreas Hegenberg</a> delivers regular feature additions and macOS compatibility updates — BetterTouchTool has been actively developed for over a decade with features driven by its large and engaged power user community.

## 💬 FAQ

<details>
<summary>Does BetterTouchTool work on Apple Silicon?</summary>

Yes. Current versions of BetterTouchTool support Apple Silicon Macs natively.

</details>

<details>
<summary>Does BetterTouchTool conflict with macOS default gestures?</summary>

BetterTouchTool works alongside macOS default gestures. Gestures assigned in BetterTouchTool can override macOS defaults if configured to do so, or can use gesture combinations that macOS doesn't use by default.

</details>

<details>
<summary>Is BetterTouchTool a one-time purchase?</summary>

BetterTouchTool offers a license purchase and is also available through the Setapp subscription. Check the BetterTouchTool website for current licensing and pricing options.

</details>

<details>
<summary>Can BetterTouchTool control window size and position through gestures?</summary>

Yes. Window resizing and repositioning actions are available as gesture triggers — assigning specific gestures to resize windows to halves, thirds, or custom sizes.

</details>

<details>
<summary>Does BetterTouchTool support haptic feedback for custom gestures?</summary>

Yes. Haptic feedback configuration is available for custom gestures to provide consistent physical feedback with standard macOS gesture feel.

</details>
